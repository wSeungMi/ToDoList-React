## ✨ Features

- 할일 추가
- 할일 수정
- 할일 삭제
- 할일 목록 조회
- 할일 완료
- 페이지네이션
- 현재 날짜 및 요일 표시

## 📑 커밋 컨벤션

```markdown
Feat : 새로운 기능 추가
Fix : 버그 수정
Design : UI 디자인 변경(CSS 스타일링)
Refactor : 리팩토링 작업 진행(코드/코드 스타일 변경 포함)
Docs : 문서 변경
Chore : 패키지 설치 및 환경 설정 변경
Rename : 파일 혹은 폴더명 수정
Remove : 파일 혹은 폴더 삭제
```
